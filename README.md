### About
`The blueprint for next-generation documents`

### Features
- Provides dynamic documentation.
- Images or videos can be set as backgrounds. See also: [Screenshots](#screenshots)
- Code is displayed with color syntax highlighting thanks to hljs.
- Images can be embedded within the documentation.
- Provides a ready-made template. See also: [File Structure](#file-structure)
- Ensures an organized working environment. See also: [File Structure](#file-structure)
- Offers a pleasant appearance. See also: [Screenshots](#screenshots)
- Provides a RoadMap to ensure disciplined work.

### Purpose
Data's Book on Demo is a fork of the documentation I developed under the Data's Books project. The purpose of this project is to serve as a template for those who want to develop documentation on any programming language or topic.

### Installation
```bash
git clone https://github.com/DATA35DELETE/Data-s-Book-on-Demo.git
```

### File Structure
```bash
Data s Book on Demo # Main folder
│   hljsLICENSE # Hljs License
│   README.md # English README
│   READMEtr.md # Turkish README
│   roadMap.md # Roadmap
│
├───Bibliography # References
│   └───example.html # Folder named after the .html file
│           depressiveData.png # .png used in example.html
│           STARFALL.mp4 # .mp4 used in example.html
│
├───Lessons # Pages/Lessons
│       example.html # Example page
│       Table Of Contents.html # Table of contents section
│
├───Media # Media files used
│   ├───image # Photos
│   │       depressiveData.png
│   │
│   └───video # Videos
│           STARFALL.mp4
│
└───Sources # Resources
    │   favicon.png  # Page icon
    │
    ├───CSS # CSS files
    │       main.css
    │
    └───JS # JS files
        │   setBackgroundMedia.js
        │
        └───hljs # Hljs files
                highlight.min.js
                vs2015.css
```

#### Bibliography (References)
Contains videos, images, or PDF files if necessary, used in HTML files.

#### Lessons (Pages)
This is where HTML files are located.

#### Media
Contains commonly used media files.

#### Sources
Contains JavaScript, CSS, and icons used.
