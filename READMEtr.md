### Özellikleri
- Dinamik bir dökümantasyon sunar.
- Arka plana resim veya video konulabilir. Ayrıca bkn: [Ekran Görüntüler](#ekran-görüntüleri)

- Hazır bir taslak sunar. Ayrıca bkn: [Dosya Yapısı](#dosya-yapısı)

### Amaç
Data's Book on Demo, Data's Books kapsamında geliştirdiğim dökümantasyonun bir fork'dur. Bu projenin amacı, herhangi bir programlama dili veya bir konu üzarine dökümantasyon geliştirmek istiyenler için bir taslaktır.

### İndirme
```bash
git clone https://github.com/DATA35DELETE/Data-s-Book-on-Demo.git
```

### Dosya Yapısı
```bash
Data s Book on Demo # Ama klasör
│   hljsLICENSE # Hljs'ın Lisansı
│   README.md # İngilizce README
│   READMEtr.md # Türkçe README
│   roadMap.md # Yol haritası
│
├───Bibliography # Kaynakçalar
│   └───example.html # Hangi .html ise onun ismi klasör olur
│           depressiveData.png # example.html de kullanılan .png
│           STARFALL.mp4 # example.html de kullanılan .mp3
│
├───Lessons # Sayfalar/Dersler
│       example.html # Örnek sayfa
│       Table Of Contents.html # İçindekinler kısmı
│
├───Media # Kullanılan medya dosyaları
│   ├───image # Fotoğraflar
│   │       depressiveData.png
│   │
│   └───video # Videolar
│           STARFALL.mp4
│
└───Sources # Kaynaklar
    │   favicon.png  # Sayfaların ikonu
    │
    ├───CSS # CSS dosyaları
    │       main.css
    │
    └───JS # JS dosyalaru
        │   setBackgroundMedia.js
        │
        └───hljs # Hljs'ın dosyaları
                highlight.min.js
                vs2015.css
```

#### Bibliography (Kaynakça)
Html dosyalarında kullanılan videolar, resimler veya gerekirse pdf dosyaları bulunur.

#### Lessons (Dersler/Sayfalar)
Html dosyaların bulunduğu kısımdır.

#### Media (Medya)
Genel olarak kullanılan medya dosyaları bulunur.

#### Sources (Kaynaklar)
Kullanılan; JavaScript, Css ve ikonlar bulunur.
